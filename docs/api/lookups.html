Available property types

GET
https://dev1.immosurance.net/yiiapp/backend/web
/import/get-property-types

Get list of available property types to be used in import file
If parent_key is null, then it's a main property type (type_one)
If parent_key has a value, then it's a secondary property type (type_two) of the main type identified by parent_key

Available property types › query Parameters

  • pagenumber

    Page number to return (pagination is 200) - Default : 1

  • searchstring

    Key to be searched with

Available property types › Headers

  • X-Api-Keystring · required

    The X-Api-Key header is used to authenticate with your API key.

Available property types › Responses

Successful operation

  • successboolean · required

    Indicates if the request was successful

  • pageinteger · required

    Current page number

  • pageSizeinteger · required

    Number of records per page

  • totalRecordsinteger · required

    Total number of records available

  • totalPagesinteger · required

    Total number of pages available

  • dataobject[] · required
